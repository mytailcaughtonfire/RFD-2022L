_This is a fork of Roblox Freedom Distribution that aims to add in a December 8th, 2022 Rōblox Studio and Client!_

⚠ This is made with the use of LLMs! 

While I do know how to code fairly well, unfortunately my expertise diminishes in server-client applications, and complicated math.

However, I do not blindly push code, I do take my time to review and see what the ai-slop code is doing. 

Please check out https://github.com/Windows81/Roblox-Freedom-Distribution to know how to run this project from source.

_Studio is not fully usable, the draggers and selectors do not work._

## Requirements
In your _GameConfig.toml_, you'll need to set the version:
```
[game_setup]
roblox_version = '2022'
```
